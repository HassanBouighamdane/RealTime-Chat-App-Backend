# Real-Time Chat App Backend
## Overview
*This repository* contains the backend implementation for a real-time chat application developed by Hassan Bouighamdane, Boutoumit El Mehdi and Boutou Chaima. 
The backend is responsible for managing user authentication, message storage, and facilitating real-time communication between users.
## Features
**User authentication**: Allows users to sign up, log in, and authenticate their sessions.  

**Real-time messaging**: Supports real-time communication between users through chat rooms or direct messages.  

**Message storage**: Stores chat messages securely and efficiently for retrieval.

## Technologies Used
**Programming Language**: Node.js.

**Web Framework**: Express.js.

**Database**:  MongoDB

**Real-time Communication between clients and the server**: WebSockets.
